一、注册账号

图片路径：images/zuce_01.jpg

注册链接：https://prod.digital-scroll.org/#/pages/reg?c=182N17

  1. 选择账号类型：手机号或邮箱（勾选其一）。
  2. 填写账号信息：
    • 输入账号（手机号或邮箱）。
    • 设置登录密码（需输入两次确认）。
  3. 填写邀请码：输入推荐人提供的邀请码（例如：182N17）。
  4. 验证码：根据右侧算式输入答案（示例中算式为0*7=?，需输入0）。
  5. 提交完成注册。

二、设置资金密码与保存安全码

图片路径：images/anquanma_02.jpg

  1. 登录后进入【我的】页面，系统会弹出"保存安全码"提示。先点击红色字体【资金密码未设置？】。
  2. 设置6位数字资金密码（例如：123456）。
  3. 获取安全码：
    • 输入资金密码点击获取安全码，系统弹出"保存安全码"提示（示例安全码：JC2zLBvtXH）。
    • 截屏或复制保存安全码（用于找回密码，切勿泄露）。
  4. 勾选【我已保存好安全码】完成验证。
重要提示： 安全码是找回账户的唯一凭证，请务必妥善保存。丢失后将无法找回账户！

三、完成实名认证

图片路径：images/shiming_03.jpg

  1. 联系推荐人授权认证： 点击你的账号进入个人信息页面，复制卷界地址发送给推荐人授权认证，等推荐人授权后，账户余额显示0.12DSC才可以认证。
  2. 进入实名认证：
    • 点击【我的】→【未实名】。
    • 填写真实姓名、身份证号等信息提交。【采集人脸】完成人脸识别。最后点击【人证对比】。
  3. 领取福利：
    • 实名通过后，联系推荐人，领取专属福利。（本福利为推荐人私人扶持，不定时更改或取消）
    • 推荐人将赠送3台卷轴矿机，系统赠送1台卷轴矿机，总计4台。

四、日常操作与领取收益

图片路径：images/chaozuo_04.jpg

  1. 每日领取收益：
    • 进入【我的】页面→ 查看"卷轴收益"和"推广收益"。
    • 点击红色数字领取（每天至少领取一次，48小时未领取将停产）。
  2. 查看个人资产信息：
    • 在【我的】页面中查看当前DSC可用余额、质押数量及收益详情。点击【活跃度】或【DSC质押】可查看所有已有卷轴。
  3. 更新APP：
    • 进入【系统设置】→ 【当前版本】下载最新版APP，确保功能正常。

注意事项

  1. 安全码与密码：安全码和资金密码是账户核心凭证，丢失可能导致资产无法找回。
  2. 实名认证：必须使用本人真实信息，否则无法通过审核。
  3. 收益领取：每天操作避免停产，建议设置每日提醒。
  4. 推荐人联系：完成注册后，添加推荐人微信，获取后续支持。

友情提示：

  • 注册时需填写正确邀请码（如182N17）。
  • 安全码页面需复制或截屏保存（示例：JC2zLBvtXH）。
  • 人脸识别时需保持整个头部在识别框内，根据提示做相应的动作，不要有多余的动作，避免逆光拍摄。
