DSC注册与挖矿图文教程

一、注册账号

注册链接：https://prod.digital-scroll.org/#/pages/reg?c=182N17

  1. 选择账号类型：手机号或邮箱（勾选其一）。
  2. 填写账号信息：
    • 输入账号（手机号或邮箱）。
    • 设置登录密码（需输入两次确认）。
  3. 填写邀请码：输入推荐人提供的邀请码，必填（例如：182N17）。
  4. 验证码：根据右侧算式输入答案（示例中算式为0*7=?，需输入0）。
  5. 提交完成注册。

二、设置资金密码与保存安全码

  1. 登录后进入【我的】页面，系统会弹出"保存安全码"提示。先点击红色字体【资金密码未设置？】。
  2. 设置6位数字资金密码（例如：123456）。
  3. 获取安全码：
    • 输入资金密码点击获取安全码，系统弹出"保存安全码"提示（示例安全码：JC2zLBvtXH）。
    • 截屏或复制保存安全码（用于找回密码，切勿泄露）。
  4. 勾选【我已保存好安全码】完成验证。
重要提示： 安全码和资金密码是账户核心凭证，丢失可能导致资产无法找回。

三、完成实名认证

  1. 联系推荐人授权认证可快速实名： 点击你的账号进入个人信息页面，复制卷界地址发送给推荐人授权认证，等推荐人授权后，账户余额显示0.12DSC才可以认证。你也可以不联系推荐人，每天领取，等领取到余额有0.12DSC就可以实名了。
  2. 进入实名认证：
    • 点击【我的】→【未实名】。
    • 填写真实姓名、身份证号等信息提交。【采集人脸】完成人脸识别。最后点击【人证对比】。
  3. 领取福利：
    • 实名通过后，联系推荐人，进入社区群聊，一起共识。（最好是保存推荐人联系方式，遇到问题可及时联系！）
注意： 必须使用本人真实信息，否则无法通过审核。

四、日常操作与领取收益

  1. 每日领取收益：
    • 进入【我的】页面→ 查看"卷轴收益"和"推广收益"。
    • 点击红色数字领取（每天至少领取一次，48小时未领取将停产）。
  2. 查看个人资产信息：
    • 在【我的】页面中查看当前DSC可用余额、质押数量及收益详情。点击【活跃度】或【DSC质押】可查看所有已有卷轴。
  3. 保存网页：
    • 把网页保存书签，以免丢失。再把网页发送到桌面就和APP一样了。APP现已暂停更新，等国际版上线后一起更新。
收益领取提醒： 每天操作避免停产，建议设置每日提醒。

注意事项

  1. 安全码与密码：安全码和资金密码是账户核心凭证，丢失可能导致资产无法找回。
  2. 实名认证：必须使用本人真实信息，否则无法通过审核。
  3. 收益领取：每天操作避免停产，建议设置每日提醒。
  4. 兑换SCC：领取收益需要燃烧SCC，建议兑换充足的SCC。
  5. 联系推荐人：完成注册后，添加推荐人微信，获取后续支持。
友情提示：
  • 注册时需填写正确邀请码（如182N17）。
  • 安全码页面需复制或截屏保存（示例：JC2zLBvtXH）。
  • 人脸识别时需保持整个头部在识别框内，根据提示做相应的动作，不要有多余的动作，避免逆光拍摄。
  • 兑换、复投卷轴系统次日会赠送SCC，领取收益时会燃烧SCC。
